## vscode 需安装插件

- ESLint
- Prettier - Code formatter
- EditorConfig for VS Code

## vscode 添加配置

```json
{
  "editor.formatOnSave": true, //  保存时格式化代码
  "editor.defaultFormatter": "esbenp.prettier-vscode" // 使用 prettier 进行格式化
}
```
